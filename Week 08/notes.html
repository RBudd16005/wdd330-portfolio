Chapter 08: Transforms and Transitions

The CSS3 transform property can rotate, scale, and/or skew any element on the page.
Elements can be manipulated using transform functions.
Translation functions can move elements left, right, up, or down.
These functions are like the behavior of position: relative; when declaring top and left,
moving elements up and down or left and right along the x and y axes.
The scale(x,y) function scales an element by the defined factors horizontally then vertically.
If only one value is provided, it will be used for both the x and y values. The rotate() function rotates an element around the point of origin by a specified angle value.
As with scale, by default the point of origin is the elementâ€™s center.
The skew(x,y) function specifies a skew along the x and y axes.
The x specifies the skew on the x axis, and the y specifies the skew on the y axis.
The transition-duration property sets how long the transition will take: the duration of time it takes to go from the default state to the transitioned state.
The transition-timing-function can control the pace of the transition in even more granular detail.
CSS animations, unlike transitions, allow you to control each step of an animation via keyframes.

Chapter 12: Canvas, SVG, and Drag and Drop

The first step to using canvas is to add a canvas element to the page.
The canvas element takes both a width and height attribute, which must also be set.
The context is the place where the drawing is rendered.
Both strokeStyle and fillStyle are set on a context object, and both take one of three values: a string representing a color, a CanvasGradient object, or a CanvasPattern object.
